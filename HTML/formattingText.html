Basic Template

Paragraphs

Here, I can write paragraphs. Any amount of information, but it's solely text. Nothing will be styled unless I use CSS to change the font, the font size, color, etc.

Headings

I can have different types of headings. The Title of this webpage is with h1 heading. This is the largest size. We can go up to h6 heading sizes.

Bold/Italics

we use "strong" to bold words. We can also italicize in two ways. One way is to use the i element tag or the em tag. Em gives emphasis and shows the urgency of the italicized word.

Lists

Lists can be ordered and unordered. An ordered list will produced list items with numbers while unordered lists will have bullets. Each item must be surrounded by li (or list item) tags. Example:

  • flour
  • egg
  1. flour
  2. egg

Quotes

We can also have things called blockquotes. This would surround the multiple lines of text that are quotes (even if held in paragraph format). For example, this paragraph is assumed to be a large block quote and we will cite it using the cite tag. Megana C The q attribute will place quote marks around your regular quotes. Additionally if your paragraph element has a language attribute attached, the q element will use the respective syntax for the quotation marks.
