Chapter 5: Images and Graphics


Images

Images element have 4 necessary attributes

  1. src = "Image.jpg"
  2. alt = "describe picture"
  3. width = "400"
  4. height = "400"


Image File/Format

Must be a file that the browser can easily access and understand. Goal is to have highest quality with smallest size of file.
Types of images

  • Gif
  • SVG (scalable vector graphic) - logos, icons
  • JPG - compressing photographs
  • PNG - sometimes better than GIF and JPG at compression


Responsive Image

Image size will change based upon the screen size. Different files can be delivered based upon the size of the hardware. An option is to save multiple resolutions of the same image and tell the browser to pick the respective image according to the criteria given. Keep the same image tag code as before and add attribute "srcset" and list all the possible images available for browser to choose from.

<img src="flower.jpeg" alt="lotus" width="400" height="400" srcset = "absolute or relative paths of remaining pics, each followed by commas">


Responsive Width

Add a sizes attribute and allow browser to choose the image based upon width of hardware <img src="flower.jpeg" alt="lotus" width="400" height="400" srcset = "absolute or relative paths of remaining pics, each followed by commas" sizes = "(max-width: 480 px) 240 px", (max-width: 960 px) 480p">
